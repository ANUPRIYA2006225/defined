Difference between Undefined and not defined variables in JS

  • Undefined variables works like when we declared a variable in the code but did not assign the value before printing the variable value but the not defined variables works like when we did not declare the variable and try to call that variable.
  • If the variable name which is being accessed doesn't exist in the memory space, then it is called not defined.

  • If the variable name which is being accessed exists in the memory space but hasn't been assigned any value till now, then it is called undefined.
